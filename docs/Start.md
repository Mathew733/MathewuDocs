## Komendy

!>Komend __Moderacyjnych__ może użyć **każdy** kto ma permisje do usuwania wiadomości!              
Dlatego ważna jest zmiana permisji komend w ``Ustawienia serwera->integracje->Mathewu``
- **<h3>Administracyjne</h3>**
  - ``/nick_bot <nick>``
  - ``/wiad <wiadomość>``
- **<h3>Moderacyjne</h3>**
  - ``/UserInfo <user>``
  - ``/warn``
  - ``/mute``
  - ``/unmute``
  - ``/kick``
  - ``/ban``
  - ``/scamban``
  - ``/unban``
